# Bank Smart Search - Lit Web Component

A fully reusable, accessible, keyboard-friendly smart search component for banking platforms.

## Features
- Web Component (Lit)
- Keyboard navigation
- Highlighting
- Dynamic positioning
- Fully themable
- Events for parent apps

<bank-smart-search></bank-smart-search>
